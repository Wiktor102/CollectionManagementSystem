# System zarządzania kolekcjonerstwem

Aplikacja została wykonana w ramach szkolnego zadania. Służy do zarządzania kolekcjami i ich zawartością. Umożliwia porządkowanie elementów, definiowanie własnych pól oraz szybki podgląd stanu całej kolekcji.

## Obsługiwane funkcje

- tworzenie i edycja kolekcji
- dodawanie, edycja i przeglądanie elementów kolekcji
- własne kolumny i pola niestandardowe dla kolekcji
- przypisywanie statusu, ceny, oceny, komentarza i obrazu do elementu
- podsumowanie kolekcji z podstawowymi statystykami
- eksport i import danych kolekcji
- zarządzanie kolejnością i zawartością kolumn
